códigos do Git


FEITOS SÓ NA INICIALIZAÇÃO

git init = iniciliza o repositório
git add /nome do arquivo/ = adiciona um arquivo na visão do git pra que possa ser comitado
git config user.email elensantanna1011@gmail.com = configura email do usuário
git config user.name "Elen Sant'Anna Filipeto" = configura o autor


ROTINEIROS

git status = mostra o status do repositório no momento
git commit -m "comentário" = cria um snapshot do repositório a cada alteração, podendo adicionar comentários explicando as alterações
